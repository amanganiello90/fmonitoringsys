Benvenuto nel sistema di monitoraggio cartelle

E' un microservizio con lo scopo di monitorare il contenuto della cartella del progetto "src/main/resources/checkFolder"

Sono esposte due web REST api:

  1. /fmonitoringsys/api/filesinfo?start=YOUR_START_DATE_TIME&end=YOUR_END_DATE_TIME : restituisce tutte le informazioni dei file contenuti nella cartella di controllo, che ricadono nell'intervallo delle dateTime
  2. /fmonitoringsys/api/file?md5=YOUR_MD5_CHECKSUM : restituisce le informazioni del file contenuto nella cartella monitorata, che corrisponde alla stringa MD5 passata

Attenzione alle dateTime : il formato deve essere yyyy-mm-ddThh:mm:ss

Di seguito esempi con risultato ottenuto dalla ricerca:

  1. http://localhost:8080/fmonitoringsys/api/filesinfo?start=2015-09-26T01:30:00.000&end=2030-09-26T01:30:00.000
  2. http://localhost:8080/fmonitoringsys/api/file?md5=d8c2984c1312a1ed8a483357f0caff5f"

Di seguito esempi con nessun risultato ottenuto dalla ricerca:

  1. http://localhost:8080/fmonitoringsys/api/filesinfo?start=2015-09-26T01:30:00.000&end=2016-09-26T01:30:00.000
  2. http://localhost:8080/fmonitoringsys/api/file?md5=sfc4984c1g12g1hd8j48y65ef0caff5
